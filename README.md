# Homebrew Tap

Personal Homebrew tap for [pszypowicz](https://github.com/pszypowicz).

## Install

```bash
brew tap pszypowicz/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| `audio-guard` | Prevents Bluetooth devices from hijacking the default macOS microphone |

### audio-guard

```bash
brew install audio-guard
brew services start audio-guard
```
