# Homebrew Tap

Personal Homebrew tap for [pszypowicz](https://github.com/pszypowicz).

## Install

```bash
brew tap pszypowicz/tap
```

## Casks

| Cask | Description |
|------|-------------|
| `mic-guard` | Prevents Bluetooth devices from hijacking the default macOS microphone |

### mic-guard

```bash
brew install --cask mic-guard
```
